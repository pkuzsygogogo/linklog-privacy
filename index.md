 # LinkLog Privacy Policy                                                     
                                                                               
  **Last updated:** 2026-04-22                                                 
                                                                               
  LinkLog is a Chrome extension for page-level note taking and visit tracking  
  during web browsing.                                      
  This Privacy Policy explains what data LinkLog processes, how it is stored,  
  and how users can control it.                                                
  
  ## 1. What data LinkLog processes                                            
                                                            
  LinkLog may process the following data created through extension usage:      
                                                            
  - Page visit metadata (for example: first visited time, last visited time,   
  visit count)                                              
  - Page-level note drafts and note history content                            
  - Optional page rating data                                                  
  - Extension settings (for example: theme, card scale, retention period,      
  reminder preferences, shortcut-related preferences, card position/pinned     
  state)                                                                       
                                                                               
  ## 2. Where data is stored                                                   
  
  LinkLog does **not** use a custom developer backend server for user content. 
                                                            
  Data is stored using Chrome extension storage:                               
                                                            
  - `chrome.storage.local` (local device storage)                              
  - `chrome.storage.sync` (Chrome account sync storage, when available)
                                                                               
  ## 3. Why data is used                                                       
                                                                               
  Data is used only to provide LinkLog features, including:                    
                                                            
  - Showing visit context on pages                                             
  - Saving and displaying notes/history                     
  - Exporting selected history to CSV                                          
  - Applying user preferences and settings                                     
  - Enabling cross-device sync behavior supported by Chrome storage            
                                                                               
  ## 4. Data sharing and transfer                                              
                                                                               
  LinkLog does not sell personal data.                                         
  LinkLog does not share note content with advertising networks.
  LinkLog does not transmit user note data to a custom external server         
  controlled by the developer.                                                 
                                                                               
  ## 5. User control and deletion                                              
                                                            
  Users can control and delete data at any time from within the extension,     
  including:                                                
                                                                               
  - Clearing history for the current page                                      
  - Clearing all extension data
                                                                               
  Users may also remove the extension to stop further data processing by       
  LinkLog.                                                                     
                                                                               
  ## 6. Permissions usage                                                      
  
  LinkLog currently uses these Chrome permissions:                             
                                                            
  - `storage`: store notes, visit metadata, and settings                       
  - `tabs`: support extension interactions such as shortcut-triggered behavior
  and opening extension-related pages                                          
  - Host permissions for `http://*/*` and `https://*/*`: enable the extension
  card and page-level behavior on visited web pages                            
                                                            
  ## 7. Policy updates                                                         
                                                            
  This Privacy Policy may be updated to reflect product or compliance changes. 
                                                            
  The “Last updated” date above indicates the latest revision.                 
                                                            
  ## 8. Contact                                                                
                                                            
  For privacy-related questions or requests, please contact the developer      
  through the support/contact channel provided on the extension listing page.
